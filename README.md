
## This is blog Pizza Application Using MERN Stack . 🍕

# Features 🚡
1. Signin and Signup.
2. authentication.
3. Cart Page.
4. Orders Page
5. Managing State using Redux
6. Auto calculations price
7. Payment Getway using Strip
8.  Relations between Multipal collections  

# Technologies used 💻
1. react 
2. Redux
3. CSS 
4. Javascript
5. react-Bootstrap
6. Express
7. Mongodb
8. Nodejs
9. Mongoose
